# Vaishnavi Misal — Portfolio

A modern, animated personal portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Three.js. This project showcases a polished presentation of skills, projects, education, experience, certifications, and contact information in a visually rich experience.

## ✨ Highlights

- Modern and responsive one-page portfolio experience
- Smooth animations and motion-driven UI
- 3D-inspired visuals and interactive effects
- Sections for About, Skills, Projects, Experience, Certifications, and Contact
- Resume download and recruiter-oriented quick access features
- Optimized for fast performance and clean presentation

## 🛠️ Tech Stack

- Next.js 13
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- Lucide Icons
- Netlify deployment support

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js 18+
- npm or pnpm

### Installation

```bash
git clone https://github.com/Vaishnavi252-ai/vaishnavi_portfolio.git
cd vaishnavi_portfolio
npm install
```

### Run locally

```bash
npm run dev
```

Then open http://localhost:3000 in your browser.

## 📦 Available Scripts

```bash
npm run dev      # Start the development server
npm run build    # Create a production build
npm run start    # Start the production server
npm run lint     # Run ESLint
npm run typecheck # Run TypeScript type checks
```

## 📁 Project Structure

```text
app/             # App router pages and metadata
components/     # Portfolio sections and UI components
lib/            # Utility helpers
public/         # Static assets
```

## 🎨 Customizing the Portfolio

You can easily personalize the portfolio by updating:

- Profile and bio content in the relevant section components under [components](components)
- Project details in [components/Projects.tsx](components/Projects.tsx)
- Resume download link in [components/ResumeDownload.tsx](components/ResumeDownload.tsx)
- Contact details in [components/Contact.tsx](components/Contact.tsx)
- Global metadata in [app/layout.tsx](app/layout.tsx)

## 🌐 Deployment

This portfolio is prepared for deployment on Vercel find that in the About Section. A production build can be generated using:

```bash
npm run build
```
